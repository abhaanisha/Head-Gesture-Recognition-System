1. PROJECT TITLE & DESCRIPTION
--------------------------------------------------------------------------------
Title: Head Gesture Recognition System — TinyML-Based Assistive Communication
       for Elderly & Disabled Individuals

Description:
A real-time, cloud-free wearable gesture recognition system that translates
head movements into meaningful feedback using two Arduino Nicla Vision boards
mounted on a cap. The system fuses 12-axis IMU data from bilateral temples,
extracts 113 statistical and cross-IMU features over a 20-sample sliding
window, and runs an m2cgen-exported Decision Tree entirely on-device in
MicroPython — with no cloud dependency. Recognised gestures are displayed on
an OLED screen via an Arduino UNO R4 WiFi, announced via Windows TTS, and
pushed as notifications to a caregiver's phone through the ntfy app.
